pywinauto使用方法
发表于 2020-04-09| 更新于 2020-04-10|python
阅读量:|评论数:

pywinauto是一组python模块来自动化Microsoft Windows GUI。最简单的是，它允许您将鼠标和键盘操作发送到Windows对话框和控件。

安装pywinauto库：
pip install pywinauto

使用pywinauto库:
1.导入pywinauto包
from pywinauto. application import Application。

2.启动一个应用程序，用start()方法：
Application().start(“路径”) start方法的参数是应用程序的路径

3.连接已运行的应用程序，用connect()方法:
例如：
app = Application().connect(process=””) process: 应用程序的进程ID
app = Application().connect(class_name=””) class_name: 类名
app = Application().connect(title=””) title: 标题文本
也可以指定窗口参数的任意组合，例如app = Application().connect(title = “”,class_name = “”)。这些参数传递给pywinauto.findwindows.find_elements()函数，根据传入的条件查找元素。
connect()方法的参数可以查看pywinauto.findwindows.find_elements()函数的参数。
查看参数地址如下：
https://pywinauto.readthedocs.io/en/latest/code/pywinauto.findwindows.html#pywinauto.findwindows.find_elements

这里介绍下 Inspect.exe 这是微软创建的一个工具。可以查看窗口及其控件的元素属性。
只要鼠标移动到哪里，它就可以显示其属性。使用这个软件可以寻找到connect()方法中的参数值。同时下述的指定对话框和调用控件这两个步骤也要用到这个软件来查找元素属性。

注意：应用程序必须准备就绪，然后才能使用connect()方法。所以在使用start()方法之后，你需要time.sleep()几秒，来等待应用程序完全启动。

4.指定应用程序的对话框：
当连接到已运行的应用程序后，需要匹配到它的窗体。
例如：dlg = app.window(title=””,class_name=””)
其方法参数可以查看pywinauto.findwindows.find_windows()的参数
地址如下：
https://pywinauto.readthedocs.io/en/latest/code/pywinauto.findwindows.html#pywinauto.findwindows.find_windows

5.调用控件：
按以下内容定位一个控件：

  1. 控件的标题，即title
  2. 控件的友好类名，即friendly class
  3. 控件的标题加类名，即title+friendly class
    例如;dlg[‘title’’friendly class’]
    可以从下列地址查看各个控件的友好类名friendly class：
    https://pywinauto.readthedocs.io/en/latest/controls_overview.html#
    如果控件的文本title为空，我们寻找最接近的文本，并附加友好类名来确定
  4. 友好类名
  5. 最近的文本+友好类名
    例如 dlg[‘最近的文本’’friendly class’]
    注意：如果字符串是中文的，需要在前面加个u。 例如 u’账号’

6.常规控件的基本操作：
例如：
一般来说按钮、复选框和组合框是有标题title的
对按钮进行点击 dlg[‘控件标题’’Button’].Click()
对复选框进行点击 dlg[‘控件标题’’CheckBox’].Click()
选中组合框的第n-1个选项 dlg[‘控件标题’‘ComboBox’].Select(n)

    编辑框这种是没有title的，例如输入账号和密码的这种框
    dlg[u‘账号’+’Edit’].SetText(‘’)        清空账号编辑框的文本
    dlg[‘账号’+’Edit’].TypeKeys(‘W9730102’)    输入编辑框的文本
    注意：输入账号密码时，最好隔零点几秒输入一个字符，如果密码中的字符同时输入，可能导致登陆出错。

其它操作以及其它控件可以查看下述网址
https://pywinauto.readthedocs.io/en/latest/controls_overview.html#

具体例子：

Application().start(ur’F:\软件\Wind\Wind.NET.Client\WindNET\bin\WindNET.exe’) # 启动应用程序
time.sleep(3) # 睡眠三秒，让程序完全启动
app = Application().connect(title=u’Wind资讯金融终端..2hai’, class_name=’#32770’) # 根据标题文本和类名连接到应用程序
time.sleep(1)
dlg = app.window(title=u’Wind资讯金融终端..2hai’) # 根据标题文本指定对话框

dlg[u’账号’’Edit’].SetText(‘’) # 清空账号编辑框
for i in ‘123456’: # 每隔0.5秒输入账号的一个字符
dlg[u’账号’’Edit’].TypeKeys(i) # 根据最近的文本和友好类名确定控件，输入账号
time.sleep(0.5)

dlg[u’密码’’Edit’].SetText(‘’) # 清空密码框
for j in ‘123456’: # 每隔0.5秒输入密码的一个字符
dlg[u’密码’’Edit’].TypeKeys(j) # 根据最近的文本和友好类名确定控件，输入密码
time.sleep(0.5)

time.sleep(3) # 暂停三秒
dlg[u’自动登录并进入首页’’CheckBox’].click() # 根据文本和友好类名点击复选框
time.sleep(3)
print(dlg[‘ComboBox’].ItemCount()) # 输出组合框的选项总数
dlg[u’选择登录服务器’’ComboBox’].Select(3) # 根据标题文本和友好类名，选中复选框的第四个选项
time.sleep(3)

dlg[u’X’’Button’].Click() # 点击窗口右上角的X按钮，关闭窗口。
